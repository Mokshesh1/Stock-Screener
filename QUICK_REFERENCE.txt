## 🔧 TROUBLESHOOTING

**CSV not found?**
- Screener still works (fallback list)
- Place file in same folder as screener

**Stocks not loading?**
- Check CSV has `symbol` column
- All symbols uppercase
- Proper format: SYMBOL,NSE/BSE,Active,Stock

**Too slow?**
- Speed = internet, not stock count
- Use daily timeframe instead of minute
- Filter by volume > 1M shares
