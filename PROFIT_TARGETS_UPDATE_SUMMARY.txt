## How It Works

### Formula:
```
profit_target = base_profit + (stock_volatility × 0.5)

Then apply caps:
- Minimum: 2% (safety floor)
- Maximum: 15-30% (depends on timeframe)
```

### Example: RELIANCE with 25% Volatility

## Technical Details

### Volatility Calculation:
```python
def calculate_volatility(df: pd.DataFrame) -> float:
    returns = df['close'].pct_change().dropna()
    volatility = returns.std() * np.sqrt(252)  # Annualized
    return max(0.02, min(volatility, 0.50))    # Capped
```

### Profit Target Calculation:
```python
def calculate_dynamic_profit_target(...):
    base = timeframe_config['base_profit']
    max_p = timeframe_config['max_profit']
    adj = volatility * 0.5
    
    profit_pct = min(max_p, max(base + adj, 0.02))
    target_price = current_price * (1 + profit_pct)
    
    return target_price, profit_pct * 100, holding_days
```

## Frequently Asked Questions

**Q: Why isn't my target showing the new %?**
A: Restart Python. Changes take effect on new run.

**Q: Can I add more timeframes?**
A: Yes! Edit `TIMEFRAMES` dict in TradingConfig class.

**Q: What if a stock has crazy volatility?**
A: Targets are capped at timeframe maximum (15-30%).

**Q: Should I use aggressive or conservative targets?**
A: Match your risk tolerance and time availability.

**Q: Why do different stocks have different targets?**
A: Volatility varies. Higher volatility = higher targets.

**Q: Can I mix strategies?**
A: No, one CURRENT_TIMEFRAME at a time. Change as needed.
